Приложение для изучения японского языка. Содержит уроки, тесты и шкалу прогресса. Разработано на Java в Android Studio.

Функциональность
Уроки по базовым темам (алфавит, приветствия, числа, фразы)
Тесты для проверки знаний после каждого урока
Шкала прогресса, которая заполняется при успешном прохождении теста

Установка

Скачайте файл debug_app.apk из репозитория, откройте его на устройстве Android и разрешите установку из неизвестных источников.
